# Margin

## [Magical Tree](https://cssbattle.dev/play/45)

<img width="400px" height="300px" loading="lazy" src="https://cssbattle.dev/targets/45.png">

#### Solution:

```html
<div></div>
<style>
  * {
    box-sizing: border-box;
  }
  body {
    background: #1a4341;
    margin: 0;
  }
  div {
    width: 90px;
    height: 120px;
    box-shadow: 0px 0px 0 30px #998235, 0px 0px 0 60px #1a4341,
      0px 0px 0 90px #f3ac3c;
    margin: 0 auto;
  }
  div::before,
  div::after {
    content: "";
    position: absolute;
    background: #998235;
    bottom: 30px;
    left: 65px;
    width: 270px;
    height: 30px;
  }
  div::after {
    background: #f3ac3c;
    width: 100%;
    top: 90px;
    left: 50%;
    transform: translateX(-50%) rotate(90deg);
  }
</style>
```
